# メモ帳アプリ (Memo App)

ブラウザのローカルストレージを利用して動作する、シンプルなシングルページアプリケーション（SPA）風のメモ帳アプリです。
フレームワークは使用せず、Vanilla JavaScript (素のJavaScript) と HTML / CSS で構築されています。

## ✨ 主な機能

- **メモの作成・保存**:
  - タイトルを入力して新しいメモを作成できます。
  - 本文は詳細画面で編集し、保存ボタンを押すことでブラウザに保存されます。
- **メモの閲覧・編集**:
  - ホーム画面のリストからメモを選択し、タイトルと本文の編集が可能です。
  - **未保存警告機能**: 編集状態で保存せずに「戻る」ボタンを押した際、誤って変更を破棄しないよう確認ダイアログが表示されます。
- **お気に入り機能とフィルター**:
  - 重要なメモに「☆ お気に入り」マークを付けることができます。
  - ホーム画面のタブを切り替えることで、お気に入り登録したメモだけを絞り込んで表示できます。
- **メモの削除**:
  - 不要になったメモは詳細画面から削除可能です（誤操作防止のため確認ダイアログが表示されます）。
- **セキュリティ対策**:
  - リストプレビュー表示時などに `escapeHtml` 処理を行い、クロスサイトスクリプティング（XSS）対策を施しています。

## 🛠 技術スタック

- **フロントエンド**: HTML5, CSS3, Vanilla JavaScript
- **データ永続化**: ブラウザの `localStorage` API
- **開発ツール / 環境**:
  - TypeScript (設定あり / `package.json` 及び `tsconfig.json`)
  - ESLint / Prettier (コードのフォーマットと静的解析用)
  - Node.js / npm (開発用パッケージ管理)

## 🚀 使い方 (起動方法)

ローカル環境で動作させるために特別なサーバーを立てる必要はありません。
プロジェクト内の `index.html` をお使いのウェブブラウザ（Chrome, Safari, Edgeなど）で直接開くだけで、アプリをご利用いただけます。

## 📁 主要なファイル構成

- `index.html` : アプリケーションのUI構造を定義するベースファイル
- `style.css` : アプリの見た目を整えるスタイルシート
- `app.js` : メインのロジック（状態管理、画面切り替え、ローカルストレージとのデータのやり取り）
- `src/` : TypeScript等の開発ソース格納用ディレクトリ
- `package.json` : 開発用パッケージやスクリプト（ビルド、Lintなど）の定義
